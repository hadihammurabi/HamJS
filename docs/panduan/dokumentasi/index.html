# Apa Itu HamJS?

HamJS merupakan kerangka kerja (framewok) NodeJS untuk membangun sebuah aplikasi berbasis web. HamJS menggunakan struktur atau konsep MVC (Model-View-Controller) yang merupakan salah satu struktur yang banyak digunakan.

Dengan mengusung konsep yang sudah banyak digunakan, diharapkan framework ini dapat digunakan dengan sangat mudah.
